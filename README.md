# SpeedGeist

Practice and speedrunning tools for **TetherGeist**, built with **LibreGeist**.

## Features

* Save States (F5)
* Load States (F9)
* Next Room (F1)
* Previous Room (F2)
* Saves player position, velocity, room, and Azae stock

## Requirements

* Aurie
* YYToolkit v5
* LibreGeist

## Installation

1. Download the latest release.
2. Place `SpeedGeist.dll` in:

```text
mods/Geist/
```

3. Launch the game.

## Controls

| Key | Action        |
| --- | ------------- |
| F5  | Save State    |
| F9  | Load State    |
| F1  | Next Room     |
| F2  | Previous Room |
